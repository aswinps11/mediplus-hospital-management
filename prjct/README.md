# MediPlus Hospital Management System

## Description
A hospital management web application developed using Django that allows users to view hospital services, doctors, and availability, book appointments, and submit contact queries. Admin users can manage doctors, appointments, and messages through a secure dashboard.

## Features
- View hospital services and doctor details
- Doctor availability management
- Appointment booking system
- Contact form
- Admin panel for managing doctors, appointments, and messages
- User authentication

## Technologies Used
- Django
- Python
- HTML
- CSS
- Bootstrap
- SQLite / MySQL

## How to Run
1. Clone the repository
2. Install requirements
3. Run migrations
4. Start the server
